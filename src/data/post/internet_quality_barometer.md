The Internet Quality Barometer (IQB), designed and developed under [Measurement Lab](https://www.measurementlab.net/), transforms raw Internet measurement data into actionable insights about Internet quality. Specifically, IQB maps raw speed test measurements to network requirements (e.g., throughput, latency), maps these requirements to representative Internet use cases (such as video streaming or web browsing), and finally aggregates performance across use cases into a single IQB score. The IQB score is a composite index ranging from 0 to 1, intended to capture overall Internet quality in a way that is both interpretable and comparable across locations.
